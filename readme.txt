The program description.

This program is the complex tool for Adobe Flash movies.
It lets you convert an Adobe Flash movie to a standalone movie.
It also lets you extract an Adobe Flash movie from a standalone movie.
A standalone movie is an executable file. It combines the built-in player and a Flash movie in one file.

The program usage.

This software is a text interface/console program.
Give a standalone movie name to decompile it.
Give the Flash Player Projector file name and the Flash movie file name to compile the standalone movie.

The additional software.

You need a Flash Player projector to compile the movie. You can take it here: https://archive.org/details/flashplayer_standalone_projectors

Exit codes.

0: The operation was successfully completed.
1: Can't open the input file.
2: Can't create the output file.
3: Can't jump to the target offset.
4: Can't allocate memory.
5: The executable file was corrupted.
6: The Flash movie was corrupted.
7: The standalone movie was corrupted.
